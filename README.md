# FUTURE_DS_03
Power BI dashboard analyzing marketing campaign conversion and customer segmentation using Bank Marketing dataset.
📊 Marketing Funnel & Conversion Performance Dashboard

This project presents a comprehensive marketing campaign analysis using Power BI. The dashboard evaluates conversion performance, customer segmentation, and campaign effectiveness using the Bank Marketing dataset.

🎯 Objective

The goal of this project is to:

Measure overall campaign conversion rate

Analyze customer behavior by age and job

Compare performance across contact methods

Identify seasonal subscription trends

Visualize drop-off using a marketing funnel

📈 Key Metrics

Conversion Rate: 19.11%

Total Contacts: 293

Total Subscriptions: 56

📊 Key Insights

Customers aged 30–45 show the highest subscription rate.

Cellular contact method performs better than telephone.

Certain months generate higher subscription volumes.

Management and technician job categories contribute significantly.

Noticeable drop-off between contacted customers and successful subscriptions.
